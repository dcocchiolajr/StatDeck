WHAT IS STATDECK?
A customizable hardware monitoring display using a Raspberry Pi 4 with a 7"
touchscreen, connected to a Windows PC via USB. Features real-time stats,
customizable tile layouts, and touch interaction.

GETTING STARTED:
================================================================================

WINDOWS SETUP:
1. Install Python 3.9+
2. cd Windows/StatDeck.Service
3. pip install -r requirements.txt
4. Edit config.json (set your COM port)
5. python main.py

RASPBERRY PI SETUP:
1. Install Raspberry Pi OS
2. Configure USB gadget mode (see docs/PI_SETUP.md)
3. Install Node.js 18+
4. cd Pi/backend && npm install
5. npm start
6. Open http://localhost:3000 in browser

- Node.js 18+
- Express (HTTP server)
- ws (WebSocket)

SUPPORT:
================================================================================

This is foundation code. You'll need to:
1. Set up your hardware
2. Follow setup guides
3. Test and iterate
4. Build the Windows Config App (or edit JSON manually)

The modular design makes it easy to:
- Add new tile types (one file)
- Add new data sources (one collector)
- Add new actions (one action handler)
- Customize styling (CSS only)
